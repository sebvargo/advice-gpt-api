Welcome to Advice Persona's documentation!
==========================================

**Advice Persona** gives advice as different personas through a simple API. 
It leverages `Advice Slip <https://api.adviceslip.com/>`_ and `ChatGPT <https://openai.com/blog/chatgpt/>`_ to provide its service. 

.. note:: This project is under active development
